## Installation

```bash
sudo pacman -S rust
```

**Check if installed:**
```bash
rustc --version
```


## Uninstalling Rust
If installed using `rustup` script

`rustup self uninstall`

This will uninstall all Rust toolchains and data, and remove `$HOME/.cargo/bin` from your PATH environment variable.
